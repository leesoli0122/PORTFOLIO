# 업무 기록 도구 (DailySummary)

> 배포 URL: https://leesoli0122.github.io/PORTFOLIO/DailySummary/
> 문서 목적: 업무 중 배운 내용과 질문을 프로젝트별로 기록하는 개인 도구 정리

---

## 프로젝트 개요

- **목적**: 작업 중 배운 내용, 질문, 해결 방법을 프로젝트 단위로 정리
- **사용 환경**: PC 브라우저 기준 정적 웹 페이지
- **기술 스택**: HTML / CSS / JavaScript
- **데이터 방식**: `data/records.json` 파일을 불러오고, 수정한 내용은 JSON으로 내보내기
- **주요 사용 상황**: 하루 작업 내용 정리, 질문/답변 보관, 프로젝트별 기록 분류

이 프로젝트는 공부한 내용이나 실무 중 질문했던 내용을 잊지 않기 위해 만든 기록 도구입니다.  
서버 없이 정적 웹 환경에서 사용할 수 있도록 JSON 파일을 불러오고 내보내는 방식으로 구성했습니다.

---

## 작업 범위와 AI 활용

전체 화면 레이아웃, 사이드바, 기록 카드, 입력 모달 UI는 직접 방향을 잡고 구성했습니다.  
JavaScript의 JSON 불러오기/내보내기, 검색, 필터, 수정/삭제 같은 데이터 처리 기능은 AI를 참고해 구현했습니다.

### 직접 작업한 부분

- 사이드바와 본문 영역 레이아웃 구성
- 기록 카드, 검색창, 상단 버튼, 입력 모달 UI 작성
- 프로젝트별로 기록을 보기 좋게 나누는 화면 흐름 정리
- CSS 파일을 `reset`, `default`, `style`로 분리
- 기록 작성에 필요한 입력 항목과 안내 문구 정리

### AI 도움을 받아 구현하거나 정리한 부분

- `records.json` 파일을 불러오는 비동기 처리
- 기록 데이터 정규화와 날짜 기준 정렬
- 전체/오늘/프로젝트별 필터 처리
- 검색어 기준 목록 렌더링
- 기록 추가, 수정, 삭제 로직
- JSON 가져오기/내보내기 기능
- 저장되지 않은 변경 상태 표시

## 현재 구현된 기능

### 1. 기록 목록

- `records.json` 데이터를 불러와 날짜 기준으로 정렬
- 같은 날짜의 기록을 그룹으로 묶어서 표시
- 오늘, 어제 라벨을 별도로 표시
- 기록이 없을 때 빈 상태 메시지 출력

### 2. 프로젝트 필터

- 기록에 입력된 프로젝트명을 기준으로 사이드바 메뉴 자동 생성
- 전체 기록, 오늘 기록, 프로젝트별 기록으로 필터링
- 프로젝트별 기록 개수 표시

### 3. 검색

- 제목, 프로젝트명, 내용 전체를 대상으로 검색
- 현재 선택된 필터 상태 안에서 검색 결과 표시
- 검색어 입력 시 목록을 다시 렌더링

### 4. 기록 추가/수정/삭제

- 모달에서 제목, 프로젝트, 날짜, 내용 입력
- 새 기록 추가와 기존 기록 수정에 같은 모달 사용
- 기록 삭제 기능
- 변경된 데이터가 있을 때 저장 안 된 변경 상태 표시

### 5. JSON 가져오기/내보내기

- 로컬 JSON 파일을 불러와 현재 기록 데이터로 사용
- 수정한 기록을 JSON 파일로 내보내기
- 정적 웹 환경이라 직접 파일을 덮어쓰지는 않고, 내보낸 JSON을 `data/records.json`로 교체하는 방식

## 아직 보완할 점

- 정적 웹 방식이라 브라우저에서 `records.json`을 직접 덮어쓸 수는 없습니다.
- 기록이 많아질 경우 페이지네이션이나 월별 필터가 필요할 수 있습니다.
- 현재 데이터 구조는 단순해서 태그, 중요도, 첨부 같은 확장 기능은 없습니다.
- JSON import/export 흐름은 AI 도움을 받아 구현했기 때문에, 파일 읽기와 다운로드 생성 흐름을 더 설명할 수 있게 정리할 예정입니다.

---

## 배운 점

- 서버가 없어도 JSON 파일을 기준으로 간단한 기록 도구를 만들 수 있다는 걸 배웠습니다.
- 데이터 렌더링은 원본 배열, 필터 조건, 검색어를 분리해 생각하면 흐름을 이해하기 쉽습니다.
- 정적 웹에서 저장 기능을 만들 때는 사용자가 파일을 직접 교체해야 한다는 한계도 함께 안내해야 한다는 점을 알게 되었습니다.

## 한 줄 소개

레이아웃과 기록 UI는 직접 구성했고, JSON 처리와 검색/필터 기능은 AI 도움을 받아 구현하며 학습했습니다.
