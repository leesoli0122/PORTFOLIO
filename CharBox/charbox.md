# 특수문자 치트시트 (CharBox)

> 배포 URL: https://leesoli0122.github.io/PORTFOLIO/CharBox/
> 문서 목적: 퍼블리싱 작업 중 자주 쓰는 특수문자와 HTML 엔티티를 빠르게 찾기 위한 개인 도구 정리

---

## 프로젝트 개요

- **목적**: 특수문자, HTML 엔티티, 맥 단축키를 한 화면에서 검색하고 복사하는 치트시트
- **사용 환경**: PC 브라우저 기준 정적 웹 페이지
- **기술 스택**: HTML / CSS / JavaScript
- **데이터 방식**: `symbol-data.js` 배열 데이터를 화면에서 필터링
- **주요 사용 상황**: 마크업 작성 중 특수문자, 화살표, 공백 문자, 저작권 기호 등을 빠르게 확인할 때

작업 중 자주 찾게 되는 특수문자를 매번 검색하지 않기 위해 만든 작은 도구입니다.  
특수문자 자체와 HTML 엔티티를 함께 볼 수 있게 구성했고, 필요한 값은 버튼으로 바로 복사할 수 있습니다.

---

## 작업 범위와 AI 활용

이 프로젝트는 HTML 테이블 구조, 검색 영역, 카테고리 UI, 전체 화면 스타일은 직접 구성했습니다.  
JavaScript의 검색, 필터, 복사 로직은 AI를 참고해 구현했고, 이후 코드 흐름을 읽으면서 필요한 문구와 데이터 구조를 수정했습니다.

### 직접 작업한 부분

- 특수문자 치트시트 화면 구성
- 검색 input, 카테고리 버튼, 테이블 구조 작성
- `reset.css`, `style.css` 스타일 정리
- 특수문자 데이터 항목 구성 방향 정리
- 화면에서 필요한 문구와 카테고리 분류 수정

### AI 도움을 받아 구현하거나 정리한 부분

- 검색어 기준으로 테이블을 다시 그리는 JavaScript 로직
- 카테고리 버튼 클릭 시 필터링하는 이벤트 처리
- 동적으로 생성되는 복사 버튼의 이벤트 위임 처리
- 클립보드 복사와 fallback 복사 로직
- HTML 문자열 삽입 시 사용하는 `escapeHtml()` 처리

## 현재 구현된 기능

### 1. 특수문자 목록 표시

- 특수문자, 설명, 카테고리, 맥 단축키, HTML 엔티티를 테이블로 표시
- 공백 문자는 화면에서 보이지 않기 때문에 별도 표시 문자로 구분
- 데이터는 `symbolList` 배열을 기준으로 렌더링

### 2. 검색

- 검색어 입력 시 목록을 즉시 다시 렌더링
- 검색 대상은 특수문자, 설명, 카테고리, 단축키, HTML 엔티티
- 영문 검색 시 대소문자 영향을 줄이기 위해 비교 문자열을 소문자로 변환

### 3. 카테고리 필터

- 전체, 공백, 화살표, 문장부호, 저작권/상표, 수학, 통화, 괄호, 기호 분류
- 선택한 카테고리와 검색어 조건을 함께 적용
- 카테고리 버튼은 부모 요소에 이벤트를 연결하는 이벤트 위임 방식으로 처리

### 4. 복사

- 특수문자 복사와 HTML 엔티티 복사 분리
- 최신 브라우저에서는 `navigator.clipboard.writeText()` 사용
- 실패하거나 지원하지 않는 경우 임시 `textarea`를 이용해 복사 재시도

## 아직 보완할 점

- 데이터가 늘어나면 카테고리 버튼을 HTML에 직접 추가하는 방식보다 데이터 기반으로 자동 생성하는 방식이 더 편할 수 있습니다.
- 복사 버튼 문구가 길어 테이블 폭이 좁을 때 답답해질 수 있어 버튼 UI를 더 줄일 필요가 있습니다.
- 현재는 개인용 치트시트라 검색 히스토리나 즐겨찾기 기능은 없습니다.
- JavaScript 로직은 AI를 참고했기 때문에, 면접 대비용으로 이벤트 위임과 클립보드 API 흐름을 직접 설명할 수 있게 정리할 예정입니다.

---

## 배운 점

- 테이블 데이터도 배열로 분리하면 검색과 필터 기능을 붙이기 쉽다는 걸 배웠습니다.
- 동적으로 생성되는 버튼은 각각 이벤트를 붙이기보다 부모 요소에서 이벤트 위임으로 처리할 수 있다는 걸 알게 되었습니다.
- 클립보드 기능은 최신 API만 믿기보다 실패했을 때의 fallback 처리를 함께 고려해야 한다는 점을 경험했습니다.

## 한 줄 소개

퍼블리싱 작업 중 자주 쓰는 특수문자와 HTML 엔티티를 검색하고 복사할 수 있는 개인용 치트시트입니다.  
화면 구조와 스타일은 직접 구성했고, 검색/필터/복사 기능은 AI 도움을 받아 구현하며 JavaScript 흐름을 학습했습니다.
