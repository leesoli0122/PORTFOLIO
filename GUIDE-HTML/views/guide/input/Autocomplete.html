Autocomplete

자동 완성

텍스트 입력 중 가능한 옵션을 드롭다운으로 제안해주는 컴포넌트입니다. jQuery UI autocomplete()를 기반으로 구현합니다.

기본 사용법

기본 검색

<input id="ac-basic" placeholder="입력해보세요" autocomplete="off" />
$("#ac-basic").autocomplete({
            source: ["React", "Vue", "Angular", "Svelte", "Solid"],
            minLength: 1,   // 최소 입력 글자 수
            delay: 200       // 입력 후 제안까지 딜레이(ms)
        });

서버 데이터 연동

기본 검색

<input id="ac-basic" placeholder="입력해보세요" autocomplete="off" />
<div class="spinner" id="remote-spinner" style="display:none"/>
$("#myInput").autocomplete({
                                source: function(request, response) {
                                  $.ajax({
                                    url: "/api/search",
                                    data: { term: request.term },
                                    success: function(data) {
                                      response(data); // 배열 or [{label, value}] 형태
                                    }
                                  });
                                },
                                minLength: 2
                              });

주요 Props (옵션) jQuery UI Autocomplete의 자주 쓰는 옵션들

옵션 타입 기본값 설명
source Array | String | Function — 제안 데이터. 배열, URL 문자열, 또는 비동기 함수로 전달
minLength Number 1 제안이 표시되기 위한 최소 입력 글자 수
delay Number 300 마지막 키 입력 후 제안 표시까지의 딜레이(ms)
disabled Boolean false true일 때 자동완성 비활성화
select Function null 항목 선택 시 콜백. ui.item.value로 선택값 접근
focus Function null 항목에 포커스 이동 시 콜백
open Function null 드롭다운이 열릴 때 콜백
close Function null 드롭다운이 닫힐 때 콜백
